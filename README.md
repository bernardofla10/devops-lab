# DevOps Lab

This repository documents my hands-on DevOps learning path.

The goal is to build practical foundations in:

- Linux operations
- Shell scripting
- Docker
- Docker Compose
- CI/CD with GitHub Actions
- Security gates with Semgrep and Trivy
- Basic cloud deployment
- Observability fundamentals
- Infrastructure as Code fundamentals

## Why this repository exists

I am preparing for a DevOps/Platform Engineering internship and using this repository to document practical labs, commands, mistakes, debugging steps and small projects.
