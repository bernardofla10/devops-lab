# Day 01 - Linux Terminal Basics

## Goal

Start using Linux as a DevOps working environment and document the first terminal commands.

## Key ideas

Linux is important for DevOps because most servers, containers and cloud workloads run on Linux.

The terminal is the main interface for:

- navigating files
- reading logs
- managing processes
- testing network connectivity
- running scripts
- debugging services

## Commands I practiced

- `pwd`
- `ls`
- `cd`
- `mkdir`
- `touch`
- `cat`
- `cp`
- `mv`
- `rm`
- `grep`
- `find`
- `ps`
- `top`
- `curl`
- `ping`
